## ADDED Requirements

### Requirement: 事业机会微页管理展示复制路径入口
系统必须在「极友料/事业机会/事业机会-微页管理」的父级微页记录操作栏提供复制路径入口，用于打开复制路径弹窗。

#### Scenario: 打开复制路径弹窗
- **WHEN** 用户在父级微页记录操作栏点击复制路径入口
- **THEN** 系统展示标题为「复制路径」的弹窗

#### Scenario: 记录缺少微页 ID
- **WHEN** 用户尝试复制缺少微页 ID 的记录路径
- **THEN** 系统不得生成无效路径，并必须给出不可复制提示或阻止打开弹窗

### Requirement: 复制路径弹窗展示双端路径
系统必须在复制路径弹窗中展示 APP 与营销助手小程序两条只读路径，路径必须基于当前微页配置 ID 生成。

#### Scenario: 展示 APP 路径
- **WHEN** 复制路径弹窗打开且微页 ID 为 `123`
- **THEN** APP 行必须展示 `/packages/mixHome/pages/microPage/index?configId=123`

#### Scenario: 展示营销助手小程序路径
- **WHEN** 复制路径弹窗打开且微页 ID 为 `123`
- **THEN** 营销助手小程序行必须展示 `/pages/groupOperation/microPage?configId=123`

#### Scenario: 路径不可编辑
- **WHEN** 用户查看复制路径弹窗
- **THEN** 两条路径输入框必须为只读或禁用状态，不允许用户直接编辑

### Requirement: 用户可复制任一端路径并收到反馈
系统必须允许用户分别复制 APP 路径和营销助手小程序路径，并必须在复制成功或失败时展示明确反馈。

#### Scenario: 复制成功
- **WHEN** 用户点击任一路径行的「复制路径」按钮且剪贴板写入成功
- **THEN** 系统必须提示「复制路径成功」

#### Scenario: 复制失败
- **WHEN** 用户点击任一路径行的「复制路径」按钮且剪贴板写入失败
- **THEN** 系统必须提示「复制路径失败」

#### Scenario: 关闭弹窗
- **WHEN** 用户点击弹窗关闭按钮
- **THEN** 系统必须关闭复制路径弹窗且不改变列表筛选状态
