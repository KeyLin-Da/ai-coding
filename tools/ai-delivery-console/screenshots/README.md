## 🎨 截图规范

- **分辨率**：建议 1920x1080 或更高，导出时宽度控制在 1400-1600px
- **格式**：PNG（无损压缩）
- **命名**：使用 `序号-功能描述.png` 格式
- **隐私**：确保截图中不包含敏感信息（如密钥、个人隐私数据）
- **清晰度**：文字清晰可读，避免过度缩放
- **主题**：建议使用浅色主题，保持视觉一致性
